Main

Overview

RDFCoder is a flexible library meant to convert the Java code, both source or bytecode, to an RDF representation.

RDF is a set of standards conceived to represent, store and query graphs expressing complex models, and it is related to the SPARQL language, a powerful query language meant to access RDF models.
RDFCoder can be used both as library and as a command-line tool.
Here somequery examples showing the flexibility of this solution.

Detailed informations on RDF and related technologies can be found here.

Purpose

How RDF Coder can be useful for you?

  • Code Analysis - Analyzing large package structures, classes, signatures and dependencies.
  • Code Refactoring - Finding library problems and relationships.
  • Custom Documentation - Generating custom documentation.
