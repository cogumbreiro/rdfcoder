Main

Overview

RDFCoder is a flexible suite of tools meant to convert Java code, both source or bytecode, to an RDF model representation.

RDF is a set of standards to represent, store and query graphs representations of complex models. RDF provides SPARQL, a powerful query language meant to inquiry RDF Graphs.The RDFCoder tools provide methods to create an RDF model of any Java library and to perform complex structure analysis.
Here some query examples.

Detailed informations on RDF and related technologies can be found here.

Purpose

How RDF Coder can be useful for you?

  • Code Analysis - Analyze package structures, class fields, methods, signatures, library dependencies.
  • Code Refactoring - Find library problems, find separation points among libraries.
  • Custom Documentation - Generate your custom documentation starting from the model.
